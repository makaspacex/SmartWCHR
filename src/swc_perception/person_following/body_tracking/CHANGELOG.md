# Changelog for package body_tracking

tros_2.0.0 (2023-05-11)
------------------
1. 更新package.xml，支持应用独立打包
2. 更新应用启动launch脚本


hhp_1.0.6 (2022-08-24)
------------------
1. 优化发布的控制指令，使机器人速度变化平缓。

------------------
